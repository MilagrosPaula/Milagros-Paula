Bitácora

Aquí encontrarás registrado el proceso de creación de esta página.

El proceso se divide en sesiones por semana, así que te invito a revisar cada sesión.

¡Disfruta el viaje!

Semana 3

En esta semana iniciaremos con la creación de nuestra página web.

Creamos una cuenta en GitHub donde subiremos nuestros archivos que contienen los recursos necesarios para tener nuestra plantilla web.

Clonamos nuestro repositorio y lo editaremos en Visual Studio Code.

CMuchos tuvimos problemas con un archivo en especial, pero después lo resolvimos cambiando el nombre del archivo y seguimos editando la página con nuestros datos.

Semana 4

En esta semana aprendimos a editar partes de la página web en específico.

Editamos los textos con nuestra información en el código en el Visual Studio Code

Aprendimos como subir y colocar otras imágenes en nuestro código.
